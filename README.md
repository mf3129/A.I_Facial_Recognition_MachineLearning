# Facial Recognition Machine Learning

This uses the Clarifai Facial Detection API.


## Install

```sh
npm install node.js
```

## Requirements
* Node.js
* Git


## Contributing
* Fork the project
* Add tests for your feature 
* Implement your feature
* Run tests again
* Commit 
* Send a pull request
